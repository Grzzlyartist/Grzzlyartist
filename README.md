## 📌 Featured Projects

### 🌾 SOYA COPILOT — AI Farming Assistant for Malawian Farmers

> AI-powered farming assistant for soybean farmers that provides real-time agricultural advice, detects infected plants using machine learning, and recommends treatment solutions to improve productivity.

**Tech Stack:** `Python` `Streamlit` `LangChain` `ChromaDB` `Google Colab` `Kaggle Datasets`

<details>
<summary>🔍 Key Features</summary>

- Real-time crop disease detection
- Personalized treatment recommendations
- Voice-enabled queries for low-literacy farmers
- Offline-capable advisory system
</details>

---

### 🧑‍🎓 DEVGet Learning Platform

> Web-based learning management system providing digital, tech, and AI education to self-paced learners. Features a real-time AI teaching assistant and community interface for course-based student interaction.

**Tech Stack:** `React` `Tailwind CSS` `Node.js` `Express` `JavaScript` `Supabase` `Render` `Vercel`

<details>
<summary>🔍 Key Features</summary>

- AI-powered teaching assistant
- Interactive coding exercises
- Student discussion forums
- Progress tracking and certificates
</details>

---

### 📊 Zeelaxia AI

> Web-based application providing AI-generated data analysis for final year students conducting research projects. Handles data cleaning, descriptive/inferential statistics, results generation, and chart visualizations.

**Tech Stack:** `Python` `React` `Node.js` `Express` `Tailwind CSS` `OpenAI API` `Firebase`

<details>
<summary>🔍 Key Features</summary>

- Automated statistical analysis (descriptive & inferential)
- Smart data cleaning and preprocessing
- Interactive chart visualizations
- APA-formatted results generation
</details>
